\subsection{Detailed Description}
converts from GA to latex for illustrating small image configurations 

{\bf Usage:} GA2tikz in.ga [b] out.tikz

{\bf Description:} Produces a LaTeX file from a binary or grayscale GA image. If {\bf b} is present, the result is binary otherwise the edges are greyscale colored.

The result should be compiled with pdflatex to produce a pdf file.

{\bf Types supported:} byte 2d

{\bf Category:} convert
