\subsection{Detailed Description}
interactive dilation / erosion by an Euclidean ball {\bfseries Usage:} \doxyref{dilatball.tcl}{p.}{dilatball_8tcl} in.pgm out.pgm

{\bfseries Description:}

Use the slide bar to adjust the radius of the ball (negative radii correspond to erosions).

Check {\bfseries combine} to see both original image and result.

Use the {\bfseries Quit} button to save and quit.
