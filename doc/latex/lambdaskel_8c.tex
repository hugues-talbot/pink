\subsection{Detailed Description}
grayscale filtered topological skeleton 

{\bf Usage:} lambdaskel in.pgm $<$imcond.pgm$|$null$>$ connex lambda out.pgm

{\bf Description:} Filtered topological skeleton for 2D grayscale images. The parameter {\bf connex} gives the connectivity used for the minima; possible choices are 4 and 8. The parameter {\bf lambda} is a contrast parameter (positive integer). Let F be the function corresponding to the input image {\bf in.pgm}. The parameter {\bf imcond.pgm} is a constraint function G. The algorithm is the following:



\footnotesize\begin{verbatim}
Repeat until stability:
    Select a point p which is lambda-destructible for F or a peak
        such that F(p) > G(p) and such that F(p) is minimal
    F(p) = alpha-(p,F)
Result: F
\end{verbatim}
\normalsize
