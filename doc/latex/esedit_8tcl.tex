\subsection{Detailed Description}
structuring element editor 

{\bf Usage:} \doxyref{esedit.tcl}{p.}{esedit_8tcl} $<$width height filename $|$ filename$>$

{\bf Description:} If given alone, parameter {\bf filename} indicates a file which contains the structuring element to be edited.

Otherwise, {\bf width} and {\bf height} indicate the dimensions of the element to be created.

Check {\bf origin} to specify the origin of the structuring element.

Check {\bf revert} to set all blue points in the structuring element, and all red points in its complementary.

Use the {\bf Abort} button to quit without saving.

Use the {\bf Save/Quit} button to save and quit.

{\bf Types supported:} byte 2d
