{\bf Usage:} hma in.pgm [mode] out.pgm

{\bf Description:} Higher-resolution Medial Axis (HMA) of the binary image {\bf Xh}, where Xh has been obtained by H(X), X being the original object and H, the H-transform.

{\bf Types supported:} byte 2d, byte 3d
