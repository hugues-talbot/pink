\section{Main Page}\label{index}
Pink is an image processing library developed at {\tt ESIEE Engineering} for research and teaching purposes. It contains implementations of over 200 algorithms for image segmentation and filtering. Most of the operators come from mathematical morphology, but it contains operators from different fields. It is a free software licensed under the {\tt Ce\-CILL } license.\subsection{Download}\label{index_Download}
The source code and the binary packages can be accessed from the download page.\subsection{Credits}\label{index_credits}
