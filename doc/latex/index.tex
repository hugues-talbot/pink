Pink is an image processing library developed at {\tt ESIEE Engineering} for research and teaching purposes. It contains implementations of over 200 algorithms for image segmentation and filtering. Most of the operators come from mathematical morphology, but it contains operators from different fields. It is a free software licensed under the {\tt CeCILL } license.\section{Download}\label{download_download}
The source code and the binary packages can be accessed from the \doxyref{Download}{p.}{download} page.\section{Mailing list}\label{index_mailing}
If you are interested in the daily life of Pink, please subscribe to the {\tt Pink Developer mailing list }. You can also post questions concerning the usage or raise issues.\section{Bug-\/hunt!}\label{index_bughunt}
If you have discovered a bug, please report it at the {\tt bug tracker}.\section{Credits}\label{index_credits}
