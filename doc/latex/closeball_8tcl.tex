\subsection{Detailed Description}
interactive closing / opening by an Euclidean ball 

{\bf Usage:} \doxyref{closeball.tcl}{p.}{closeball_8tcl} in.pgm out.pgm

{\bf Description:}

Use the slide bar to adjust the radius of the ball (negative radii correspond to openings).

Check {\bf combine} to see both original image and result.

Use the {\bf Quit} button to save and quit.

{\bf Types supported:} byte 2d
