draw line segments which are specified by a text file 

{\bf Usage:} drawline in.pgm curve.txt out.pgm

{\bf Description:} The file {\bf curve.txt} contains a list of points under the format:\par
 c nb\_\-points\par
 x1 y1\par
 x2 y2\par
 x3 y3\par
 ...\par
